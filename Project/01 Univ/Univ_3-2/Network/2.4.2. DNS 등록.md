
# 과정

1. 등록 기관에 정보를 제공
	1. 제공 정보
		1. 호스트 이름
		2. 책임  DNS 서버 IP주소
			1. 주
			2. 부
2. 등록 기관
	1. .com TLD server
		1. 주 RR
		2. 부 RR
	2. ex
		1. networkutopia.com
			1. dns1.networkutopia.com
			2. type NS
		2. dns1.networkutopia.com
			1. 212.212.212.1
			2. type A
3. 신청 기관 담당자
	1. 기관 내 책임 DNS 서버에 RR을 등록
		1. type A record
			1. www.networkuptopia.com
		2. type MX record
			1. networkutopia.com
